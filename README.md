# CampusIQ – AI Talent Discovery Platform

CampusIQ is an AI-powered institutional knowledge engine that connects
students, skills, and projects into a searchable knowledge graph.

## Features
- Student knowledge graph visualization
- Skill relationship mapping
- Project discovery
- AI-powered insights

## Tech Stack
Frontend: React + TypeScript + Vite  
Backend: Node.js + Express  
Database: PostgreSQL (Drizzle ORM)  
Visualization: Force Graph

## Demo
Run locally:

npm install
npm run dev
